思路:
1.
偿债能力量化指标(短期)
相关指标:
    流动比率=流动资产/流动负债
    速动比率=（流动资产-存货）/流动负债
    现金比率=现金类资产/流动负债

2.
确定偿债能力理论值
流动比率，速动比率，现金比率给定理论值
    2.1:
    求各个行业股票的均值作为各个指标的理论值


3.确定单项量化指标
tar1=(流动比率-理论值)/理论值
tar2=(速动比率-理论值)/理论值
tar3=(现金比率-理论值)/理论值

4.
确定偿债能力量化指标(加权)
final_tar=1/3*tar1+1/3*tar2+1/3*tar3

5.
根据final_tar进行排序
